Инструкция по установке всех компонент для запуск интернет-ресурса:
1) Установка сервера Node.js:
   Для скачивания серверной платформы зайдите по адресу: https://nodejs.org/en/download/
   и выберите операционную систему для скачивания.
   Инструкции по установке: https://learn.javascript.ru/ajax-nodejs.
   Не забудьте установить node.js глобально.
2) Когда серверная платформа установлена вместе с ней установится пакетный установщик npm
3) Найти на диске, который добавлен к диплому, папку diploma - тут находится весь исходный код ресурса.
   Если исходный каким-либо образом пострадал, то его можно заново скачать по адресу:
   https://github.com/alexisinwork/diploma и нажать кнопку "download ZIP".
4) Для того, чтобы запустить ресурс нужно сделать следующее (установка касается поьзователей Windows):
   1) Перейти в командную строку (В поиске Windows написать cmd и нажать Enter).
   2) Определить на каком диске находится папка с исходным кодом ( например диск Е):
      1) Перейти на диск используя команду - cd e:
      2) Если папка находится по такому пути: E:/Мои Файлы/Дипломы/6 курс/Студент, тогда нужно написать следующее:
        cd Мои Файлы/Дипломы/6 курс/Студент (возможно нужно будет заменить / на \)
      3) Теперь вы находитесь в папке с проектом. Осталось написать две команды: npm install
         Подождать установки всех необходимых библиотек и потом написать: npm start
      4) После этого сервер запустится и нужно будет зайти в любой любимый браузер (к примеру Google Chrome),
         и перейти по адресу http://localhost:8080. Ресурс запущен!
